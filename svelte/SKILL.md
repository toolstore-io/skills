# Svelte Skills Sub-Index

Load the specific skill for your task. Apply the ones relevant to what you are building — not all skills will apply to every project.

| Skill | Load when you are... |
|---|---|
| `svelte-component` | Creating a new UI component or reorganizing an existing one |
| `cva-variants` | Defining or modifying visual variants for a component |
| `token-first-styling` | Applying styles or reviewing whether styles use design tokens correctly |
| `bits-ui-composition` | Building interactive primitives (dialogs, menus, toggles, tabs, checkboxes) |
