# Frontend Skills Sub-Index

## Shared (framework-agnostic)

These skills apply regardless of whether the project uses Svelte, React, or another framework:

| Skill | Load when you are... |
|---|---|
| `token-first-styling` | Applying styles or reviewing whether styles use design tokens correctly |
| `cva-variants` | Defining or modifying visual variants for a component |
| `primitive-extraction` | Deciding whether to inline a style or extract a reusable primitive |
| `storybook` | Adding or reviewing a UI component |
